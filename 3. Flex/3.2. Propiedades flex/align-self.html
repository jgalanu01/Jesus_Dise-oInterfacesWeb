La propiedad align-self actúa exactamente igual que align-items, salvo que se utiliza sobre un ítem hijo específico y no sobre el elemento contenedor.

Gracias a ese detalle, align-self permite cambiar el comportamiento de align-items y sobrescribirlo con comportamientos específicos para ítems concretos que no queremos que se comporten igual que el resto.

En este ejemplo, el valor auto se comporta como baseline porque el contenedor flex tiene align-items: baseline.
