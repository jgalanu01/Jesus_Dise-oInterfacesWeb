La propiedad row-gap establece la separación entre filas si flex-direction: column o si estamos en un contenedor multilínea (flex-wrap: wrap).

La propiedad column-gap establece la separación entre columnas si flex-direction: row o si estamos en un contenedor multilínea (flex-wrap: wrap).

La propiedad gap permite establecer row-gap y column-gap al mismo tiempo.

Las tres propiedades admiten valores dados en porcentajes. Tal y como indica la especificación de W3C, cuando se indica un valor en porcentaje este se resuelve computando el tamaño del content box del contenedor. Esto lo puedes comprobar en este mismo ejemplo utilizando las DevTools del navegador. Como hemos indicado en el CSS que el contenedor sea de 200px de ancho y con un padding de 20px, la anchura del content box es de 200-20-20 = 160px. Si pones column-gap: 10% el navegador calculará el 10% de 160px, 16px, y lo aplicará.
