Examen jQuery. Ejercicio 2

2.1. Cuando el usuario pulse el botón "Generar", si en el select hay algo seleccionado y hay algún radiobutton seleccionado, se creará dinámicamente en #resultado un párrafo con el mensaje "Has seleccionado un [marca de coche] de color [color seleccionado]". Ese párrafo será del color seleccionado. Para crear el texto debes usar el atributo data-color de cada radiobutton, mientras que para darle el color al texto debes usar el value. [2 puntos]

2.2. Al pulsar sobre un párrafo creado dinámicamente, este se eliminará definitivamente del DOM. [1 punto]

2.3. Al pulsar el botón "Reset" se debe vaciar el div#resultado [1 punto]
